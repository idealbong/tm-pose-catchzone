# 게임 규칙

## 🎮 게임 제목

**Catch Zone**

---

## 📝 게임 설명

화면은 좌(LEFT), 가운데(CENTER), 우(RIGHT) 총 3개 구역으로 나뉘어져 있습니다.
플레이어는 포즈 인식을 통해 바구니의 위치를 좌/가운데/우로 이동시키며,
화면 상단에서 떨어지는 과일 아이템을 받아 점수를 얻습니다.
하지만 폭탄을 받으면 즉시 게임 오버!
과일을 2번 놓치면 게임 오버!
게임 오버 전까지 최대한 높은 점수를 획득하세요!

---

## 🎯 게임 목표

- 게임 오버 전까지 최대한 많은 과일을 수집하여 높은 점수 획득하기
- 폭탄을 피하고 과일은 놓치지 않기
- 단계가 오를수록 빨라지는 아이템 낙하 속도에 대응하기

---

## 🕹️ 게임 규칙

### 기본 규칙

1. **시작**: "Start" 버튼을 클릭하여 게임 시작

2. **바구니 제어**: 포즈 인식을 통해 바구니 위치 제어
   - 왼쪽 포즈 → LEFT 구역으로 이동
   - 중립 포즈 → CENTER 구역으로 이동
   - 오른쪽 포즈 → RIGHT 구역으로 이동

3. **아이템 수집**: 바구니와 아이템이 같은 구역에서 만날 때 획득 판정
   - 사과(Apple): +100점
   - 배(Pear): +150점
   - 오렌지(Orange): +200점
   - 폭탄(Bomb): 즉시 게임 오버

4. **게임 오버 조건**:
   - 폭탄을 받았을 때
   - 과일 아이템을 2번 놓쳤을 때

### 놓침(미스) 규칙

- 과일 아이템을 놓치면 다음 규칙이 적용됩니다:
  - 첫 번째 놓침 → "경고" 표시, 게임은 계속 진행
  - 두 번째 놓침 → 게임 오버

- 폭탄을 놓치면 미스 카운트에 포함되지 않습니다 (폭탄 회피 성공)

### 단계(Level) 시스템

- 게임은 여러 단계로 구성되며, 각 단계는 20초 동안 진행됩니다.
- 게임 오버 없이 단계 시간이 모두 지나면 다음 단계로 넘어갑니다.
- 단계가 오를수록 아이템 낙하 속도가 빨라집니다.

**단계별 떨어짐 시간 (drop time):**
- 1단계: 2초
- 2단계: 1.8초
- 3단계: 1.6초
- 4단계: 1.4초
- 이후 단계마다 0.2초씩 감소

**아이템 생성 간격:**
- 단계별 떨어짐 시간의 60%~80% 사이에서 무작위로 결정
- 예: 1단계 → 1.2초~1.6초 사이 랜덤

## 🎨 필요한 포즈 목록

게임에 필요한 모든 포즈를 나열합니다. (Teachable Machine에서 학습할 클래스)

| 포즈 이름 | 설명 | 게임 내 동작 |
|---------|------|------------|
| 왼쪽 (Left) | 몸을 왼쪽으로 기울이거나 얼굴을 왼쪽으로 돌린 상태 | 바구니가 LEFT 구역으로 이동 👈 |
| 가운데 (Center) | 정면을 바라본 중립 상태 | 바구니가 CENTER 구역에 위치 😊 |
| 오른쪽 (Right) | 몸을 오른쪽으로 기울이거나 얼굴을 오른쪽으로 돌린 상태 | 바구니가 RIGHT 구역으로 이동 👉 |

> **참고**: 각 포즈마다 Teachable Machine에서 최소 50개 이상의 샘플을 수집하는 것을 권장합니다.

## 🍎 아이템 종류

게임에 등장하는 아이템과 효과:

| 아이템 | 아이콘 | 점수 | 특징 |
|-------|-------|------|------|
| 사과 (Apple) | 🍎 | +100점 | 기본 과일 아이템 |
| 배 (Pear) | 🍐 | +150점 | 중간 점수 과일 |
| 오렌지 (Orange) | 🍊 | +200점 | 고득점 과일 |
| 폭탄 (Bomb) | 💣 | 게임 오버 | 받으면 즉시 게임 종료 |

**아이템 생성 규칙:**
- 아이템은 화면 상단의 3개 구역(LEFT, CENTER, RIGHT) 중 하나에서 무작위로 생성
- 아이템 종류도 무작위로 결정 (폭탄 출현 빈도는 조절 가능)

## ⏱️ 시간 설정

게임 시간 관련 설정:

- **단계 지속 시간**: 20초 (각 단계마다)
- **1단계 아이템 낙하 시간**: 2초
- **단계당 속도 증가**: 낙하 시간 0.2초 감소
- **아이템 생성 간격**: 낙하 시간의 60%~80% 무작위
- **최소 낙하 시간**: 0.6초 (안전 장치)

## 💡 게임 개발 팁

### Teachable Machine 모델 학습 시 주의사항

1. **3개 포즈만 명확하게 구분**
   - 왼쪽, 가운데, 오른쪽 포즈를 확실히 다르게 학습
   - 특히 가운데(중립) 포즈는 왼쪽/오른쪽과 명확히 구분되도록

2. **충분한 샘플 수집**
   - 각 클래스당 최소 50개 이상 권장
   - 다양한 각도와 거리에서 촬영

3. **자연스러운 포즈 설계**
   - 빠르게 포즈 전환이 가능하도록 설계
   - 예: 몸을 좌우로 기울이기 또는 얼굴 방향 전환

### gameEngine.js 구현 가이드

게임 로직 구현 시 필요한 주요 기능:

```javascript
// 게임 상태 관리
- currentLevel: 현재 단계
- score: 점수
- missCount: 놓친 과일 개수 (2회 초과 시 게임 오버)
- levelTimer: 단계 타이머 (20초)

// 아이템 관리
- items: 화면에 떨어지는 아이템 배열
- itemTypes: ['apple', 'pear', 'orange', 'bomb']
- itemZones: ['LEFT', 'CENTER', 'RIGHT']

// 바구니 위치
- basketZone: 'LEFT' | 'CENTER' | 'RIGHT'
- 포즈 인식 결과에 따라 업데이트

// 충돌 감지
- checkCollision(): 바구니와 아이템이 같은 구역에 있는지 확인
- handleItemCatch(): 아이템 획득 처리
- handleItemMiss(): 아이템 놓침 처리

// 단계 시스템
- getDropTime(level): 단계별 낙하 시간 계산
- getSpawnInterval(dropTime): 생성 간격 계산
- levelUp(): 다음 단계로 이동
```
